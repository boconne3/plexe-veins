The majority of the code in this folder is code that has been taken from the Plexe github repo (https://github.com/michele-segata/plexe).

The major changes to the source code are in the folder ./src/veins/modules/application/platooning/<various>.

As mentioned in the final dissertation, a unique scenario was defined in the scenarios folder at the destination mentioned above. The standard sinusoidal scenario was alos adapted to suit my purposes.

A new platoon position helpser was created in the utilities folder.

The joinAtBack maneuver files were adapted for my own use as well.


The folder ./mySimulations was the source of many results and analysis scripts. Configurations were added to each of the configuration files to facilitate varying different non-ideal conditions. Plotting scripts were adapted to suit the plotting of several different condittions.
